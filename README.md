# Get-Factors
This C script is intended to get the factors and product of said factors of any given number. The results are provided in a easily readable table.

# Example Usage
```console
$ ./a.out -8
|First factor	|Second Factor	|Product	|
|-1	          |8		        |7		  |
|1	           |-8		       |-7		 |
|-2	          |4		        |2		  |
|2	           |-4		       |-2		 |
```
